
# Smart Meter Discovery Bridge

Diese Home Assistant Integration überträgt Smart Meter MQTT-Daten von einer Quelle an ein Ziel und erstellt automatisch Home Assistant Discovery-Payloads.

## Installation (via HACS)

1. Öffne HACS → Integrationen → Benutzerdefinierte Repositories.
2. Repository-URL: `https://github.com/dein-github-user/smart_meter_discovery`
3. Typ: `Integration`.
4. Nach der Installation Home Assistant neu starten.
